## 关于
BiFang是专门为Cocos Creator快速开发而生的，专注于组件化的探索。
追求极简的使用方式，对原有系统最小的侵入

同时毕方也在收集Cocos Creator周边插件，项目，以及游戏开发提高效率的相关工具
